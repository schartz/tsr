# Tiny Software Renderer
This is a tiny software (cpu only) renderer, mostly for my own educatoinal purposes. It is able to render 3d models in **triangulated** [Wavefront .obj file format](https://en.wikipedia.org/wiki/Wavefront_.obj_file).
This codebase mostly follows the [ssloy's](https://github.com/ssloy) amazing [tinyrenderer tutorial](https://haqr.eu/tinyrenderer). 
The original completed source code of the tutorial is [in the tutorial repo](https://github.com/ssloy/tinyrenderer).
